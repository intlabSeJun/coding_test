[ 기본 설명서 ]
출결, 과제에 따라 벌금 계산기.

- 사전 vba 코드 넣어주기.
excel 열고 -> 개발도구 -> Visual Basic -> (왼쪽에) 'Microsoft Excel 개체' 우측 마우스 
-> 파일 가져오기 -> '출결계산0908.bas' 선택 -> 새로운 파일 "모듈" 만들어짐 -> 클릭, Module21이 해당 코드
-> 엑셀 시트로 돌아가서 -> 개발도구 -> 매크로 -> count_attendance 클릭 후 실행 -> 자동 계산 완료.

- 구성 및 사용법
sheet1 "출석 및 과제표" : 날짜별로 출석표 및 과제 수행여부,분량,질문 / 기타 건의사항 기제
sheet2 "벌금합계" : 각 사람마다 지각/결석한 금액 나타내고 합계와 총 잔금 및 빠져나간 돈 자동 기제됨.
출결-> sheet1에서의 출석 여부(○, △, X  #앞 2개 문자는 'ㅁ'한자, X는 대문자)에 따라서
          sheet2의 벌금 합계가 지각/결석 별로 합산됨. 
과제 -> sheet1에서의 과제 여부(5,4,3,2,1)에 따라서
          sheet2의 벌금 합계가 지각/결석 별로 합산됨. = (5-안한횟수)*1000


[ 세부 설명서 ]
vba 사용하여 코드를 구성하여서 위의 설명서 대로 하면 문제 없음.
